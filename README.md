#  E-Commerce Website Manual Testing Project

##  Project Overview
This project focuses on **manual testing** of an e-commerce website using a structured QA approach that covers login, registration, product search, cart management, and checkout flow.

- **Website Under Test:** [Demo Web Shop](https://demowebshop.tricentis.com)
- **Testing Type:** Manual Functional Testing
- **Tools Used:** Excel, Postman
- **Operating System:** Windows 10
- **Browser:** Google Chrome (Latest)

---

## Objectives
To verify that all core functionalities of the e-commerce website operate as expected, including:
- User registration and login (with validation)
- Product search
- Add to cart
- Checkout process
- Error message handling
- User Account 
- serach functionality
